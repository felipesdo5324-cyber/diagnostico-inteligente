# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/14nF7X0i94HNWMTZ86ZgUuTWBsdUQ8HzR

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a `.env.local` file and set your OpenAI API key using one of the formats below:
   - `OPENAI_API_KEY=SEU_TOKEN_DA_OPENAI`
   - or `VITE_OPENAI_API_KEY=SEU_TOKEN_DA_OPENAI`
3. Run the app:
   `npm run dev`
